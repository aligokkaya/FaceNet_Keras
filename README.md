# FaceNet_Keras
## Install

1-Install dependencies

```
pip install pickle
pip install mtcnn
pip install opencv-python
pip install numpy

```

2-Adding new persons (or images of existing persons) into a system
First, creating a directory for raw images of new persons as follow.

